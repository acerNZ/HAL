How to build Node2Cam (please build it with GCC 4.7, or you may have strange error!)

1, add /Node2Cam to RPG/HAL/Camera/Drivers/
2, when build RPG, select Node2Cam option on.
3, make -j, you will see an error about Node2Cam.
4, type make -j again and that error will gone. Done. (If you know how to solve this error, please inform luma@gwu.edu)
